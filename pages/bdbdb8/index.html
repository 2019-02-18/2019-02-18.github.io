# JS实现扫码登录

扫码登录是一种方便快捷的登录方式，用户可以通过扫描二维码来登录网站或应用程序。在本文中，我们将使用JavaScript实现一个简单的扫码登录功能。

# 步骤一：生成二维码

首先，我们需要生成一个二维码，用于展示给用户扫描。我们可以使用第三方库 qrcode.js (opens new window) 来生成二维码。以下是使用qrcode.js生成二维码的示例代码：

<!-- 在HTML中引入qrcode.js库 -->
<script src="https://cdn.bootcdn.net/ajax/libs/qrcodejs/1.0.0/qrcode.min.js"></script>

<!-- 在HTML中添加一个容器用于显示二维码 -->
<div id="qrcode"></div>

<!-- 在JavaScript中生成二维码 -->
<script>
const qrcode = new QRCode(document.getElementById("qrcode"), {
  text: "https://example.com/login", // 二维码包含的内容，这里是登录页面的URL
  width: 256, // 二维码宽度
  height: 256, // 二维码高度
});
</script>

# 步骤二：监听扫码事件

当用户扫描二维码后，我们需要将扫描结果发送到服务器进行验证。我们可以使用WebSocket来监听扫码事件，并将扫描结果发送到服务器。以下是使用WebSocket监听扫码事件的示例代码：

const socket = new WebSocket("wss://example.com/socket"); // 连接到服务器的WebSocket

socket.addEventListener("open", () => {
  // 连接成功后，向服务器发送一个扫码事件
  socket.send(JSON.stringify({ type: "scan", data: { code: "xxxxxx" } }));
});

socket.addEventListener("message", (event) => {
  const data = JSON.parse(event.data); // 解析服务器返回的消息

  if (data.type === "login") {
    // 登录成功，跳转到用户首页
    window.location.href = "https://example.com/user";
  } else if (data.type === "error") {
    // 登录失败，显示错误提示
    alert(data.message);
  }
});

# 步骤三：验证扫码结果

在服务器端，我们需要验证扫描结果的有效性，并返回登录成功或失败的消息。以下是使用Node.js实现服务器端验证扫码结果的示例代码：

const WebSocket = require("ws");
const express = require("express");
const app = express();

// 创建WebSocket服务器
const wss = new WebSocket.Server({ port: 3000 });

// 监听WebSocket连接事件
wss.on("connection", (socket) => {
  socket.on("message", (message) => {
    const data = JSON.parse(message); // 解析客户端发送的消息

    if (data.type === "scan") {
      // 验证扫码结果的有效性，这里省略验证逻辑

      // 返回登录成功消息
      socket.send(JSON.stringify({ type: "login" }));
    }
  });
});

// 启动Express服务器
app.listen(80, () => {
  console.log("Server started!");
});
